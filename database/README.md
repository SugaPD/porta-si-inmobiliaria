# Base de Datos

Esta carpeta contiene los archivos relacionados con la base de datos del sistema.

La base de datos se utilizará para almacenar la información de usuarios, propiedades, clientes y ventas del sistema.

## Sistema gestor de base de datos

- MySQL

## Contenido de esta carpeta

- Scripts SQL
- Diseño del modelo de datos
- Estructura de tablas del sistema

La base de datos será utilizada por el backend mediante Spring Data JPA y Hibernate.
