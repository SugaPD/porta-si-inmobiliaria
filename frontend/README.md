# Frontend

Esta carpeta contiene el desarrollo del frontend del Sistema de Gestión Inmobiliaria.

El frontend es la interfaz con la que los usuarios interactúan para consultar, registrar y administrar propiedades dentro del sistema.

## Tecnologías utilizadas

Framework principal
- React.js

Lenguaje de desarrollo
- JavaScript

Herramientas y librerías
- Vite
- React Router
- Axios
- Tailwind CSS

## Funciones principales del frontend

- Visualización del catálogo de propiedades
- Registro y edición de propiedades
- Panel administrativo para usuarios internos
- Comunicación con el backend mediante APIs REST

## Entorno de desarrollo

Para ejecutar el frontend se requiere:

- Node.js
- npm

Instalación básica:

npm install
npm run dev
