# Sistema de Gestión Inmobiliaria - Porta SI

Este proyecto consiste en el desarrollo de un sistema web para la gestión de propiedades inmobiliarias de la empresa Porta SI. El objetivo del sistema es centralizar la información de propiedades, facilitar su administración interna y mantener un catálogo público actualizado.

El sistema permitirá registrar, editar, consultar y gestionar propiedades, así como generar reportes y controlar el acceso de los usuarios del sistema.

## Arquitectura del sistema

El sistema se desarrollará utilizando una arquitectura web compuesta por tres componentes principales:

- Frontend (interfaz de usuario)
- Backend (lógica de negocio)
- Base de datos

## Tecnologías utilizadas

### Frontend
- React.js
- JavaScript
- Vite
- React Router
- Axios
- Tailwind CSS

### Backend
- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- Spring Security
- Hibernate

### Base de Datos
- MySQL

### Seguridad
- Autenticación mediante JWT
- Encriptación de contraseñas con BCrypt

### Control de versiones
- Git
- GitHub

### Herramientas de desarrollo
- NetBeans
- Visual Studio Code
- Swagger / OpenAPI para documentación de APIs

## Estructura del repositorio

/backend → Código del servidor y lógica de negocio  
/frontend → Aplicación web del sistema
