# Backend

Esta carpeta contiene el backend del Sistema de Gestión Inmobiliaria.

El backend es responsable de manejar la lógica de negocio del sistema, así como la comunicación con la base de datos y la seguridad de la aplicación.

## Tecnologías utilizadas

Lenguaje de programación
- Java

Framework principal
- Spring Boot

Librerías y herramientas
- Spring Web
- Spring Data JPA
- Spring Security
- Hibernate

## Funciones del backend

- Gestión de usuarios del sistema
- Registro y administración de propiedades
- Procesamiento de ventas
- Generación de reportes
- Manejo de autenticación y seguridad mediante JWT

## Seguridad

El sistema implementa:

- Autenticación con JSON Web Tokens (JWT)
- Encriptación de contraseñas con BCrypt
